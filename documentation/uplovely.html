Uplovely ​

Uplovely is a web app that allows users to quickly and easily verify the authenticity of files. In collaboration with a higher-level project based on microservices, it uses the advantages of blockchain to increase the security and transparency of files. Uplovely is an important part of this vision.

My role at Uplovely as a designer and developer ​

At Uplovely I worked as a designer and frontend developer. I was responsible for all UI/UX design to create a user-friendly and responsive user interface. This meant that I had to communicate closely with the development team to ensure that the design reflected the functionality of the product.

After the design was conceived and approved, I was responsible for implementing the project in AngularJS. As a developer, my job was to ensure that the final product was optimal, performant and scalable. Throughout the entire process, I was always focused on improving the user experience and design of the product.

The web app for secure and trustworthy file verification ​

Using Uplovely ensures the authenticity of files by checking them for authenticity and immutability. Once a file is stored on the blockchain, it can no longer be edited or manipulated. This means that any change to the file will be recognized immediately and will not be recognized as the original version. In this way, Uplovely can make an important contribution to securing confidential information, e.g. contractual agreements or official documents.

Features ​

Input Switcher ​

Uplovely allows uploading files or direct hashcode input with an easy-to-use toggle feature. This feature simplifies verification and significantly improves the user experience.

Details ​

After confirming the authenticity of a file, Uplovely displays detailed information such as the name of the file, the block number on the Ethereum blockchain, the issuer, the creation date and the hashcode.

Conclusion ​

My work at Uplovely gave me deep insights into the use of microservices and how they can improve the efficiency and scalability of web applications. It was an enriching experience to work on a project that increases security and trustworthiness in digital file sharing.
