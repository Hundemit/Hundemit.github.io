Pokedex ​

1. Einführung ​

Willkommen zur meinem Pokedéx. Entdecke Eigenschaften, Typen und Statistiken zu über 800 Pokémon durch eine intuitive und benutzerfreundliche Oberfläche, entwickelt mit React und Tailwind CSS.

2. Erste Schritte ​

Pokédex Webseite ​

Öffne janhindemit.de/app-pokedex in deinem Browser, um direkt auf den Pokedex zuzugreifen. Keine Installation erforderlich.

Download und lokale Installation ​

Das vollständige Projekt ist auf GitHub unter Hundemit/Pokedex-React verfügbar. Folge den Anweisungen im README.md für Details zur Installation und Einrichtung.

3. Technologien ​

Der Pokedex basiert auf modernen Webtechnologien, um eine schnelle, responsive und benutzerfreundliche Erfahrung zu bieten:

  • React: Eine JavaScript-Bibliothek für den Bau von Benutzeroberflächen, die es ermöglicht, reaktive und wiederverwendbare UI-Komponenten zu erstellen.
  • Tailwind CSS: Ein Utility-first CSS-Framework für maßgeschneidertes Design ohne das Schreiben von Custom CSS.
  • TypeScript: Eine auf JavaScript basierende Sprache, die Typisierung zur Kompilierzeit bietet, um die Entwicklung großer Anwendungen zu erleichtern und zu beschleunigen.

4. Benutzeroberfläche ​

Die Gestaltung der Benutzeroberfläche des Pokedex orientiert sich an einer klaren und nutzerzentrierten Designphilosophie.

Startseite ​

  • Übersicht der Funktionen: Die Startseite bietet einen schnellen Zugriff auf alle wesentlichen Funktionen des Pokedex, inklusive der Suchfunktion und einer Auswahl an hervorgehobenen Pokémon-Karten.
  • Navigation: Eine übersichtliche Menüführung erleichtert die Navigation durch die Anwendung. Benutzer können leicht zwischen verschiedenen Sektionen wechseln, ohne die Orientierung zu verlieren.

Suchfunktion ​

  • Wie man nach Pokémon sucht: Benutzer können den Namen eingeben, um spezifische Pokémon zu finden. Die Suchergebnisse werden sofort angezeigt, was die Entdeckungsreise erleichtert.

Pokémon-Karten ​

  • Beschreibung der Informationen auf den Karten: Jede Pokémon-Karte zeigt ein Bild des Pokémon, seinen Namen, Typ und wichtige Statistiken. Diese Karten sind das Herzstück des Pokedex und bieten einen schnellen Überblick über die Charakteristiken jedes Pokémon.

11. Schlusswort ​

Vielen Dank an alle, die den Pokedex nutzen!
